## ✨ 特性

- ⏰ **智能计时** - 经典25分钟专注倒计时
- 📊 **实时进度** - 动态进度条，专注进度一目了然
- ✅ **任务管理** - 添加、追踪、完成任务
- 💾 **自动保存** - 数据持久化，关闭程序不丢失
- 📈 **数据统计** - 查看专注时长和完成情况
- 🔊 **声音提醒** - 专注结束自动提醒

## 🎯 快捷键

| 按键 | 功能 |
|:---:|:---|
| `P` | 暂停计时 |
| `R` | 继续计时 |
| `Q` | 退出程序 |
| `S` | 查看统计 |

## 🚀 快速开始

### 环境要求
- .NET 6.0 SDK 或更高版本
- Windows / macOS / Linux

### 安装与运行

```bash
# 克隆仓库
git clone https://github.com/你的用户名/PomoFocus.git

# 进入项目目录
cd PomoFocus

# 运行程序
dotnet run
